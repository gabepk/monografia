  \begin{resumo}
  O \textit{2Path} é um banco de dados de redes metabólicas. Ele possui um sistema que recebe um arquivo FASTA de múltiplas sequências e verifica se as sequências de enzimas presentes no banco correspondem às sequências presentes no FASTA. Caso alguma corresponda, então essa enzima está presente no organismo em questão. O objetivo deste trabalho é apresentar um projeto de interface para o banco \textit{2Path} em que seja possível fazer \textit{upload} de arquivos FASTA que representam organismos, além de pesquisar por enzimas e vias metabólicas entre dois compostos no mesmo. Para isso, serão analisadas as interfaces dos bancos de dados de redes metabólicas KEGG, BioCyc e Reactome. A interface proposta será testada e aperfeiçoada de acordo com o Método de Avaliação de Comunicabilidade, da área de Interação Humano-Computador.
  \end{resumo}

  \selectlanguage{american}
  \begin{abstract}
  The 2Path is a metabolic network database. It has a system that receives as an input a multiple sequence FASTA file and verifies if the enzymes on the database can be produced by the uploaded organism represented by the FASTA file. If so, then the organism is linked to them on the database. The goal of this project is to propose an interface to the 2Path database such that users can upload a FASTA file and search for enzymes and metabolic pathways between two compounds in the database. To achieve this goal, the interface of the most known metabolic database (KEGG, BioCyc and Reactome) will be analysed. The proposed interface will be tested and improved according to the Communicability Evaluation Method, from the Human-Computer Interaction field.
  \end{abstract}
